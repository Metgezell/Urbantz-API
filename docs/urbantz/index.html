Urbantz API Integratie

Welkom bij de lokale Urbantz API documentatie. Deze pagina biedt een overzicht van de Urbantz API integratie en hoe je deze kunt gebruiken.

Wat is Urbantz?

Urbantz is een platform voor last-mile delivery optimalisatie. Deze integratie maakt het mogelijk om taken (pickup/delivery) aan te maken via de Urbantz API.

Setup & Configuratie

1. Environment Variables

Kopieer env.example naar .env en vul je API key in:

cp env.example .env
Belangrijk: Zorg ervoor dat je een geldige Urbantz API key hebt voordat je begint.

2. Dependencies Installeren

npm install ts-node dotenv

3. Test de Integratie

npm run urbantz:dry-run

API Reference

Authenticatie

De Urbantz API gebruikt API key authenticatie via de x-api-key header.

Kernendpoint

POST /v2/announce - Maakt een nieuwe taak aan

Verplichte velden:

  • customerRef - Klant referentie
  • deliveryAddress.line1 - Adres regel 1
  • serviceDate - Service datum (YYYY-MM-DD)

Optionele velden:

  • pickupAddress - Pickup adres
  • timeWindowStart/End - Tijdvenster (HH:MM)
  • items[] - Items array
  • notes - Extra notities

Code Voorbeelden

Basis Taak Aanmaken

import { announceTask } from './src/clients/urbantz'; const task = await announceTask({ customerRef: "CUST-001", deliveryAddress: { line1: "Hoofdstraat 123", postalCode: "1234AB", city: "Amsterdam" }, serviceDate: "2024-01-15", timeWindowStart: "09:00", timeWindowEnd: "17:00" });

Pickup & Delivery Taak

const pickupDeliveryTask = await announceTask({ customerRef: "CUST-002", pickupAddress: { line1: "Ophaaladres 456", city: "Utrecht" }, deliveryAddress: { line1: "Leveradres 789", city: "Amsterdam" }, serviceDate: "2024-01-16", items: [{ description: "Pakket", quantity: 1, weightKg: 2.5 }] });

Response Codes

  • 200 - Taak succesvol aangemaakt
  • 400 - Bad Request - ongeldige payload
  • 401 - Unauthorized - ongeldige API key
  • 409 - Conflict - taak bestaat al
Tip: Gebruik de reference.md file voor een snelle API referentie, of bekijk de src/clients/urbantz.ts voor de volledige TypeScript implementatie.
